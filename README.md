# Introduction

This project is a website designed for football fans where they can watch live matches, check results, and follow their favorite teams in a simple and fast way.

The goal is to create an accessible platform that brings all football-related information together in one place.

# Inspirations

My inspirations are:

- Platforms like ESPN and TyC Sports for match results and news
- Sports streaming websites where users can watch live matches
- Football mobile apps that provide real-time statistics

# Website Ideas

The website will include:

- 🔴 Live matches section
- 📅 Match schedule
- 🏆 Results from major leagues
- ⭐ Featured teams
- 📊 Match statistics
- 🔎 Search for teams or players

# Technologies

- HTML for structure
- CSS for design
- JavaScript for interactivity

# Objective

To provide a simple, fast, and attractive experience for users who want to follow football from any device.
